Troubleshooting

If sync isn't working:
  1. Check Google Account Sync: Settings → Sync and Google services → Make sure sync is ON for all items
  2. Check Extension in Sync: Settings → Extensions → Check if Memory Jogger is syncing
  3. Verify Same Account: Make sure both machines are signed into the SAME Google account
  4. Check Network: Both machines must be online and able to reach Google services
  5. Check Quota: 'View All Storage Data' shows total bytes used. Limit is 102,400 bytes
  6. Force Sync: Chrome doesn't have manual "force sync" but you can try:
    • Sign out and back into Google account
    • Close and reopen Chrome
    • Disable and re-enable sync
  7. Extension Health: Make sure extension is installed and enabled on both machines
