# Privacy Policy (Draft)

Memory Jogger for LinkedIn stores user-authored notes in browser local storage.

## Data collection

- The extension does not collect analytics.
- The extension does not send user notes to a remote server.

## Data storage

- Notes are stored in `chrome.storage.local` on the user's browser profile.
- Users can remove the extension to stop its operation.
